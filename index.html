Page Structure Tags

Skeleton

<html> Anything between the opening and closing tags will be interpreted as HTML code </html>
<head> Contains the metadata for a web page </head>
<body> Only content inside the opening and closing body tags can be displayed to the screen </body>
<main> Represents the dominant content of the body of a document </main>
<footer> Content at the bottom of the subject information </footer>

Divisions

<div> Short for “division” or a container that divides the page into sections </div>
<article> Holds content that makes sense on its own </article>
<section> Defines elements in a document, such as chapters, headings, or any other area of the document with the same theme </section>
<aside> Used to mark additional information that can enhance another element but isn't required in order to understand the main content </aside>

Lists

<ul> Used to create a list of items in no particular order </ul>

<ol> Like unordered lists, except that each list item is numbered </ol>

<li> Used to represent an item in a list </li>

<dl> Represents a description list </dl>

<dt> Specifies a term in a description list, must be used inside a dl element </dt>

<dd> Provides the description, definition, or value for the preceding term dt </dd>

Links

<a href= "URL"> Text for the link here, sends the user to the specefied URL </a>

<a href= #ID > Text for the link here, sends the user to the specefied section of the web page according to the entered ID </a>

Content Tags

<p> Parragraph element. Contains a block of plain text </p>

<span> Contains short pieces of text. Best to use it when you want to target a specific piece of content that is on the same line as other text </span>

<img src="URL of the image" alt="Alternative text / description of the image" />

<video src="URL of the video" poster="URL for a thumbnail for the video" controls(gives user controls over the video if included)> Text in case the video doesn't load </video>

<audio src="URL of the audio" autoplay(audio will automatically begin playing as soon as it can) controls(gives user controls over the audio if included) > Text in case the audio is not supported by the browser </audio>

Tables

Tag

Description
<table> Element that will contain all of the tabular data we plan on displaying
<thead> Section off the table's column headings
<tbody> Contains all of the table's data
<tfoot> The bottom part of a long table, often used to contain sums, differences, and other data results
<tr> Sets off each row in a table
<th> Sets off the table heade
<td> Sets off each cell in a row
